\section{背景}

\textit{From Tokens to Thoughts} \cite{shani2025} 的核心问题是：\textbf{LLM 的嵌入空间是否与人类的概念结构对齐？} 论文从两个角度切入：

\begin{itemize}[noitemsep]
  \item \textbf{RQ1（范畴对齐）：} 聚类 LLM 嵌入能否还原人类定义的类别？指标：AMI（调整互信息）。
  \item \textbf{RQ2（典型性结构）：} LLM 嵌入中的概念相似度能否预测人类对类别成员的典型性评分？指标：Spearman $\rho$。
  \item \textbf{RQ3（压缩-意义权衡）：} LLM 嵌入在信息瓶颈（Information Bottleneck）框架下的压缩-失真权衡曲线是否收敛到人类类别所在的自然拐点？指标：$\mathcal{L}$ 曲线。
\end{itemize}

我们使用 24 个模型（含 word2vec、GloVe、13 个 Qwen 变体、2 个 GPT、2 个 Gemma、2 个 Phi、4 个 BERT/RoBERTa/DeBERTa、3 个 DeepSeek-R1 蒸馏版）在 McCloskey1978 数据集的静态嵌入上进行复现。嵌入维度从 50（word2vec）到 7168（DeepSeek-R1-32B）不等。

\section{信息论基础}

本节从信息论第一原理出发，逐步推导本报告使用的全部信息论度量，为后续的实验分析提供严格的数学基础。

\subsection{熵（Entropy）}

\textbf{自信息}是信息论最基本的量。对于一个离散随机变量 $X \sim p(x)$，事件 $X=x$ 的自信息定义为：
\[
I(x) = \log_2 \frac{1}{p(x)} = -\log_2 p(x)
\]
单位为比特。自信息刻画了观察到该事件所获得的"惊讶度"——发生概率越低的事件，自信息越大。

\textbf{熵}（Shannon entropy）是自信息的期望值，描述了随机变量 $X$ 的平均不确定性：
\[
H(X) = \E_{X}\left[-\log_2 p(X)\right] = -\sum_{x \in \mathcal{X}} p(x) \log_2 p(x)
\tag{1}
\]
其中约定 $0\log_2 0 = 0$。熵具有以下基本性质：
\begin{itemize}[noitemsep]
  \item \textbf{非负性：} $H(X) \geq 0$，当且仅当 $X$ 为确定性变量时取等。
  \item \textbf{上界：} $H(X) \leq \log_2 |\mathcal{X}|$，在均匀分布时取等（最大熵原理）。
  \item \textbf{凸性：} $H(X)$ 是 $p(x)$ 的凹函数。
\end{itemize}

\textbf{联合熵}描述一对随机变量 $(X,Y) \sim p(x,y)$ 的总不确定性：
\[
H(X,Y) = -\sum_{x \in \mathcal{X}} \sum_{y \in \mathcal{Y}} p(x,y) \log_2 p(x,y)
\tag{2}
\]

\textbf{条件熵}是在已知 $Y$ 的条件下 $X$ 剩余的不确定性：
\[
H(X \mid Y) = \E_Y\left[H(X \mid Y=y)\right] = -\sum_{y} p(y) \sum_{x} p(x \mid y) \log_2 p(x \mid y)
\tag{3}
\]

三者通过\textbf{链式法则}相联系：
\[
H(X,Y) = H(X) + H(Y \mid X) = H(Y) + H(X \mid Y)
\tag{4}
\]

\subsection{互信息（Mutual Information）}

\textbf{互信息}衡量两个随机变量 $X$ 和 $Y$ 之间相互包含的信息量——即知道 $Y$ 能减少 $X$ 的不确定性的程度：
\[
I(X;Y) = H(X) - H(X \mid Y)
\tag{5}
\]

由链式法则（式~4）可得等价形式：
\[
I(X;Y) = H(X) + H(Y) - H(X,Y)
\tag{6}
\]

互信息也可以直接从联合分布和边缘分布写为\textbf{KL 散度}形式：
\[
I(X;Y) = \sum_{x,y} p(x,y) \log_2 \frac{p(x,y)}{p(x)p(y)}
       = \KL\bigl(p(x,y) \,\|\, p(x)p(y)\bigr)
\tag{7}
\]

这个形式揭示了互信息的本质：$I(X;Y)$ 衡量联合分布 $p(x,y)$ 与独立假设 $p(x)p(y)$ 之间的\textbf{KL 散度}。当 $X$ 和 $Y$ 独立时，$I(X;Y)=0$；当 $X$ 完全决定 $Y$ 时（或反之），互信息达到最大。

互信息的性质：
\begin{itemize}[noitemsep]
  \item \textbf{对称性：} $I(X;Y) = I(Y;X)$
  \item \textbf{非负性：} $I(X;Y) \geq 0$，当且仅当 $X$ 与 $Y$ 独立时取等
  \item \textbf{上界：} $I(X;Y) \leq \min\{H(X), H(Y)\}$
\end{itemize}

\subsection{数据处理不等式}

\textbf{数据处理不等式}（Data Processing Inequality, DPI）是信息论中最重要的定理之一，它约束了信息在级联处理中的流向：

\begin{theorem}[Data Processing Inequality]
  若 $X \to Y \to Z$ 构成马尔可夫链（即 $p(x,z \mid y) = p(x \mid y)p(z \mid y)$），则
  \[
  I(X;Y) \geq I(X;Z)
  \tag{8}
  \]
  换言之，任何对数据的后处理都不能增加它包含的原始信息。
\end{theorem}

本报告的实验涉及以下马尔可夫链：
\[
\text{原始文本} \to \text{LLM 嵌入} \to k\text{-means 聚类} \to \text{类别标签}
\]

DPI 保证了 $I(\text{原始文本}; \text{类别标签}) \geq I(\text{嵌入}; \text{类别标签}) \geq I(\text{聚类}; \text{类别标签})$——聚类的过程只会丢失信息而不会增加信息。这正是 RQ3 的核心：在压缩（通过 $k$-means 减少信息量）和保留意义（保留对分类有用的信息）之间取得平衡。

\subsection{信息瓶颈原理}

\textbf{信息瓶颈}（Information Bottleneck, IB）\cite{tishby2000} 是一个统一的信息论框架，用于寻找一个随机变量 $T$（称为\textbf{瓶颈变量}或表示），使得 $T$ 在尽可能压缩输入 $X$ 的同时，保留对目标 $Y$ 的预测能力：

\[
\min_{p(t \mid x)} \; \mathcal{L}_{\text{IB}} = I(X;T) - \beta I(T;Y)
\tag{9}
\]

其中：
\begin{itemize}[noitemsep]
  \item $I(X;T)$ 是\textbf{压缩项}（rate），衡量 $T$ 保留了 $X$ 中的多少信息
  \item $I(T;Y)$ 是\textbf{意义项}（distortion/ relevance），衡量 $T$ 对预测 $Y$ 的有用程度
  \item $\beta \geq 0$ 是\textbf{拉格朗日乘子}，控制压缩与意义的相对权重
\end{itemize}

当 $\beta \to 0$ 时，最优解退化为将所有 $x$ 映射到同一个 $t$（最大压缩、零意义）；当 $\beta \to \infty$ 时，最优解退化为恒等映射（无压缩、保全部信息）。IB 框架的关键洞察是：\textbf{对于给定的数据分布，存在一条由 $\beta$ 参数化的最优压缩-意义曲线}——称为\textbf{IB 曲线}（rate-distortion 曲线），每个 $\beta$ 对应曲线上一个 Pareto 最优的点。

IB 框架与 $k$-means 聚类的联系见于 \cite{slonim2000}：当 $T$ 是离散的类别变量（即聚类结果）、且失真度量采用均方误差时，IB 目标退化为经典 $k$-means 的正则化形式。这为我们使用 $k$-means 近似求解 IB 问题提供了理论依据。

\section{RQ1：范畴对齐}

\subsection{方法}

\textbf{目标：} 检验 LLM 嵌入空间的聚类结构是否与人类定义的概念类别一致。

\paragraph{输入：} 每个概念的\textbf{静态嵌入} $\mathbf{e}_i \in \mathbb{R}^d$（$d$ 为嵌入维度，取决于模型），通过对模型最后一层所有子词 token 做平均池化得到。共 $n \approx 200$ 个概念（McCloskey1978），分属 $k_{\text{human}}=18$ 个人类定义的类别（如"鸟""水果""交通工具"等）。

\paragraph{流程：}
\begin{enumerate}[noitemsep]
  \item \textbf{加载嵌入：} 读取所有概念的静态嵌入（来自 	exttt{models\_embeddings/\{model\}.npy}），形状为 $(n, d)$。
  \item \textbf{$k$-means 聚类：} 使用 scikit-learn 的 KMeans（$n_{\text{init}}=100$ 次随机初始化，取最优解）将所有 $n$ 个点聚成 $k=k_{\text{human}}=18$ 个簇。$k$-means 的目标是最小化簇内平方和（惯性）：
  \[
  \min_{\{\mu_1,\dots,\mu_k\}} \sum_{i=1}^{n} \min_{c} \|\mathbf{e}_i - \mu_c\|^2
  \]
  其中 $\mu_c$ 是簇 $c$ 的质心坐标。$n_{\text{init}}=100$ 意味着每次聚类独立运行 100 次随机初始化，选取惯性最小的结果，以降低 KMeans 陷入局部最优的风险。
  \item \textbf{评估对齐：} 将聚类得到的簇标签 $\hat{y}_i$ 与人类真实类别标签 $y_i$ 对比，计算 AMI 和 NMI。
\end{enumerate}

\paragraph{输出：} 每个模型 $\times$ 数据集组合得到一个标量 AMI $\in [0,1]$，1 表示聚类结果与人类标签完全一致，0 表示与随机分类无异。

\subsection{AMI 的严格推导}

\textbf{互信息 MI} 衡量聚类结果 $U$ 与真实标签 $V$ 之间的关联。给定 $n$ 个样本，设 $U$ 有 $r$ 个簇、$V$ 有 $c$ 个类别，定义\textbf{列联表}（contingency table）：$n_{ij}$ 为同时属于 $U$ 的簇 $i$ 和 $V$ 的类别 $j$ 的样本数；$a_i = \sum_j n_{ij}$ 为簇 $i$ 的总数，$b_j = \sum_i n_{ij}$ 为类别 $j$ 的总数。

由此构造经验概率分布：
\[
p(i,j) = \frac{n_{ij}}{n},\quad
p_U(i) = \frac{a_i}{n},\quad
p_V(j) = \frac{b_j}{n}
\]

代入互信息的定义（式~6）：
\[
\text{MI}(U,V) = \sum_{i=1}^{r} \sum_{j=1}^{c} \frac{n_{ij}}{n} \log_2 \frac{n_{ij} / n}{(a_i/n)(b_j/n)}
            = \sum_{i,j} \frac{n_{ij}}{n} \log_2 \frac{n_{ij} \cdot n}{a_i \cdot b_j}
\tag{10}
\]

式~10 是 MI 的另一种等价形式，在聚类评估中被广泛使用。然而，MI 有一个严重缺陷：即使 $U$ 和 $V$ 完全独立（随机分类），由于有限样本效应，$n_{ij}$ 的期望值 $\E[n_{ij}] = a_i b_j / n$ 未必等于所有实际观测值，因此 $\E[\text{MI}] > 0$。也就是说，\textbf{随机分类也能得到正值的 MI}。

\textbf{AMI}（Adjusted Mutual Information）\cite{vinh2009} 通过减去随机分类的期望值来校正这一偏差：
\[
\text{AMI} = \frac{\text{MI}(U,V) - \E[\text{MI}(U,V)]}{\frac{1}{2}[H(U) + H(V)] - \E[\text{MI}(U,V)]}
\tag{11}
\]

其中：
\[
H(U) = -\sum_{i=1}^{r} \frac{a_i}{n} \log_2 \frac{a_i}{n}, \quad
H(V) = -\sum_{j=1}^{c} \frac{b_j}{n} \log_2 \frac{b_j}{n}
\tag{12}
\]

分母 $\frac{1}{2}[H(U) + H(V)]$ 是 $H(U,V)$ 的上界（因为 $H(U,V) \leq H(U) + H(V)$），用其作为归一化确保 AMI $\in [0,1]$。

\textbf{期望互信息} $\E[\text{MI}(U,V)]$ 的推导需利用超几何分布：在保持 $a_i$ 和 $b_j$ 不变的条件下，$n_{ij}$ 服从超几何分布 $n_{ij} \sim \text{Hypergeom}(n, a_i, b_j)$。由超几何分布的二阶矩性质可得：
\[
\E[\text{MI}] = \sum_{i=1}^{r} \sum_{j=1}^{c} \sum_{n_{ij} = \max(0, a_i+b_j-n)}^{\min(a_i, b_j)} \frac{n_{ij}}{n} \log_2 \frac{n_{ij} \cdot n}{a_i \cdot b_j} \cdot \frac{\binom{a_i}{n_{ij}}\binom{n-a_i}{b_j-n_{ij}}}{\binom{n}{b_j}}
\tag{13}
\]

式~13 的计算量随 $n$ 增长较快，实际实现中采用 Vinh 等人\cite{vinh2009} 提供的封闭形式近似。

\subsection{RQ1 分析}

\textbf{发现 1：经典嵌入大幅领先。} word2vec（AMI=0.552）和 GloVe（0.462）远超所有 LLM。这是因为 word2vec 在训练时就以词为单位的共现统计作为目标，天然匹配词级别的分类任务；而 LLM 的上下文嵌入包含了大量句法、语用信息，这些信息在类别判别中是噪声。

\textbf{发现 2：LLM 间差异有规律但不大。} LLM 的 AMI 范围仅为 $[0.024, 0.189]$。几个趋势：
\begin{itemize}[noitemsep]
  \item \textbf{较小模型意外更好：} gemma-2-9b（0.189）> gemma-2-2b（0.096），gpt2-medium（0.178）> gpt2（0.170）——但差距不大。Qwen 系列中 0.5B 到 32B 全部在 0.045~0.120 之间。
  \item \textbf{编码器 vs 解码器：} RoBERTa/DeBERTa 等编码器并不总比等规模解码器好（roberta-large 0.125 vs Qwen2-7B 0.117，差距微弱）。
  \item \textbf{DeepSeek 系列偏低：} 三个蒸馏版均低于 0.055，可能是蒸馏导致的表示空间信息损失。
\end{itemize}

\textbf{与论文的一致性：} 论文报告 40+ 个模型 AMI 均高于随机，我们确认 24 个模型全部高于随机基线（McCloskey1978 随机期望 AMI $\approx 0.0$）。经典嵌入最优的结论也得到确认。

\section{RQ2：典型性结构}

\subsection{方法}

\textbf{目标：} 检验 LLM 嵌入空间中的概念相似度是否能预测人类对类别成员典型性（typicality）的感知——即更"典型"的成员（如"鸟"中的"知更鸟"）是否在嵌入空间中更靠近类别中心。

\paragraph{输入：} 与 RQ1 相同的 $n$ 个概念的\textbf{静态嵌入} $\mathbf{e}_i \in \mathbb{R}^d$，以及每个人类类别 $C$ 中每个概念 $i$ 对应的\textbf{人类典型性评分} $\tau_i$。典型性评分来自人类实验：被试者对"该概念在多大程度上是该类别的典型成员"打分（McCloskey1978 采用 1-10 分制）。

\paragraph{流程：}
\begin{enumerate}[noitemsep]
  \item \textbf{计算嵌入相似度：} 对于每个类别 $C$ 中的概念 $i$，计算其嵌入 $\mathbf{e}_i$ 与该类别中其余概念的平均嵌入之间的余弦相似度：
  \[
  s_i = \frac{1}{|C|-1}\sum_{j \in C, j \neq i} \cos(\mathbf{e}_i, \mathbf{e}_j)
  \tag{14}
  \]
  其中 $\cos(\mathbf{a}, \mathbf{b}) = \frac{\mathbf{a} \cdot \mathbf{b}}{\|\mathbf{a}\|\|\mathbf{b}\|}$。$s_i$ 越大，说明概念 $i$ 的嵌入更接近该类别中其他概念的平均方向——直观上就是更中心的成员。
  \item \textbf{排序对比：} 将每个类别中所有概念的 $s_i$ 从大到小排序，同时将人类典型性评分 $\tau_i$ 也从大到小排序。
  \item \textbf{计算 Spearman $\rho$：} 计算两个排序之间的 Spearman 秩相关系数 $\rho \in [-1,1]$。$\rho=1$ 表示排序完全一致（嵌入相似度最高的概念也是人类认为最典型的），$\rho=0$ 表示无关，$\rho<0$ 表示反向一致。
\end{enumerate}

\paragraph{输出：} 每个模型 $\times$ 数据集组合得到一个 Spearman $\rho$ 及其 $p$ 值。$p<0.05$ 表示相关性统计显著。

\paragraph{信息论角度：} 典型性可以理解为概念在嵌入空间中的\textbf{信息密度}。高度典型的概念位于类别嵌入空间的中心附近，描述它所需的比特数较少（压缩效率高）；边缘概念偏离中心，需要更多比特来描述。如果嵌入空间忠实地编码了这种信息密度差异，那么 $s_i$ 应与 $\tau_i$ 正相关。

\subsection{RQ2 分析}

\textbf{发现 1（典型性预测能力普遍很弱）：} 即使是最优的 LLM（gemma-2-9b，$\rho=0.197$），也只能解释不到 $4\%$ 的典型性变异（$R^2\approx 0.04$）。相比于 RQ1 中 LLM 能有效区分类别边界，RQ2 表明 LLM 嵌入几乎无法捕捉类别内部的典型性层级结构。

\textbf{发现 2（经典嵌入再次领先）：} word2vec（$\rho=0.344$）和 GloVe（0.372）的表现在统计上优于所有 LLM。这可能是因为词共现统计天然编码了典型性——更典型的成员（如\"知更鸟\"）在共现语料中更频繁地与类别名（\"鸟\"）同时出现，从而在嵌入空间中更接近类别中心。而 LLM 的上下文嵌入包含了句法角色、语用功能等额外信息，这些信息稀释了典型性信号。

\textbf{发现 3（模型大小与典型性负相关）：} Qwen 系列中，$\rho$ 随模型规模增大而下降：0.5B（$\rho=0.114$）> 32B（$\rho=0.034$，不显著）。DeepSeek 蒸馏版同样如此：7B（$\rho=0.059$）> 32B（$\rho=0.035$）。这可能是因为大模型在预训练中习得了更多的句法、语用和世界知识，这些知识与典型性无关，却占据了嵌入空间的主要维度——类似于信息论中的\textbf{率失真折衷}：更大的模型容量被用来编码更丰富的语言现象，反而削弱了对特定语义属性（典型性）的聚焦。

\textbf{发现 4（跨数据集不一致）：} 几乎所有模型在 Rosch1975 和 Rosch1973 数据集上均呈现负相关的趋势。这可能是因为这两个数据集的类别间区分度更高、类别内样本更少（Rosch1973 仅 48 个条目），导致典型性评分的统计噪声主导了相关性。

\textbf{从信息论视角的解释：} 典型性可以理解为类别条件分布 $p(x \mid y)$ 下不同样本的信息密度差异。中心成员具有高先验概率（更\"可预测\"），因此 $-\log_2 p(x \mid y)$ 较低；边缘成员的低先验概率导致较高的自信息。如果嵌入空间忠实地拟合了 $p(x \mid y)$ 的几何结构，则应观察到 $\rho > 0$。然而实验结果表明 LLM 嵌入更关注类别之间的边界（由 $I(X;Y)$ 捕捉），而非类别内部的信息密度梯度——这与 DPI 一致：聚类操作（RQ1）丢弃了大部分信息，而典型性正是这些被丢弃信息的组成部分。

\section{RQ3：压缩-意义权衡}

\subsection{信息瓶颈框架下的推导}

本报告施加一个重要简化：将瓶颈变量 $T$ 限定为 $k$-means 聚类结果 $C_k$（其中下标 $k$ 表示聚类数）。此时，输入 $X$ 为静态嵌入，目标 $Y$ 为人类类别标签。

\textbf{压缩项} $I(X; C_k)$ 的计算：对于 $k$-means 聚类，$C_k$ 是 $X$ 的\textbf{确定性函数}（每个点被分配到唯一簇），因此 $p(c \mid x) \in \{0,1\}$。由互信息的定义：
\[
I(X; C_k) = H(C_k) - H(C_k \mid X) = H(C_k)
\tag{15}
\]
因为 $H(C_k \mid X) = 0$（给定 $X$ 后 $C_k$ 完全确定）。$H(C_k)$ 取决于聚类结果的大小分布。若 $k$ 个簇大小均衡（各为 $n/k$），则 $H(C_k) = \log_2 k$；若分布不均匀，$H(C_k)$ 略小于 $\log_2 k$。因此，\textbf{压缩项随 $k$ 对数增长}：每增加一倍的簇数，约需多 1 比特的信息来描述。

\textbf{意义项} $I(C_k; Y)$ 衡量聚类结果与人类标签的匹配程度。$I(C_k; Y)$ 越大，说明聚类保留了越多的分类信息。但 $I(C_k; Y)$ 难以直接估计，因为 $p(c,y)$ 的估计需要大量样本。

\textbf{替代方案——失真项}：原论文使用\textbf{均方失真}作为意义项的代理：
\[
D(k) = \frac{1}{n}\sum_{c} \sum_{x \in c} \|x - \mu_c\|^2
\tag{16}
\]
其中 $\mu_c$ 是簇 $c$ 的质心。在信息论框架下，$D(k)$ 与 $I(C_k; Y)$ 的关系可以通过\textbf{率失真理论}（Rate-Distortion Theory）来理解。

\textbf{率失真理论}告诉我们，给定失真度量 $d(x,\hat{x})$，最小的互信息（率）$R$ 满足平均失真 $\E[d] \leq D$：
\[
R(D) = \min_{p(\hat{x} \mid x): \E[d] \leq D} I(X; \hat{X})
\tag{16}
\]

在本问题中，$\hat{X}$ 是 $C_k$ 的质心表示（即聚类重构），失真度量 $d(x, \mu_c) = \|x - \mu_c\|^2$。率失真函数 $R(D)$ 是一个凸递减函数：要降低失真 $D$，必须以增加率 $R$（即更精细的聚类）为代价。$k$-means 恰好是率失真函数的一种实际近似求解——它在给定 $k$（等价于给定率 $R \approx \log_2 k$）的条件下最小化失真。

\textbf{复合目标：}
\[
\mathcal{L}(k) = \underbrace{I(X; C_k)}_{\text{复杂度/率}} + \beta \cdot \underbrace{D(k)}_{\text{失真}}
\tag{18}
\]

这正是 IB 目标（式~9）的\textbf{对偶形式}。在 IB 框架中，拉格朗日函数 $\mathcal{L}_{\text{IB}} = I(X;T) - \beta I(T;Y)$ 寻求压缩和意义的最佳折衷。由于 $I(C_k;Y)$ 的估计困难，原论文用 $D(k)$ 代替 $-I(C_k;Y)$（注意负号：最大化意义等价于最小化失真），得到：
\[
\mathcal{L}(k) \approx I(X; C_k) + \beta D(k)
\]
其中 $\beta$ 将原 IB 中的拉格朗日乘子重新缩放为失真项的权重。我们固定 $\beta = 1.0$，此时 $\mathcal{L}(k)$ 随 $k$ 变化的形状由数据的几何结构和聚类的率失真特性共同决定。

\textbf{为什么 $k=18$ 是自然的拐点？} 由率失真理论，$R(D)$ 曲线上的每个点对应一个 Pareto 最优的压缩-失真折衷。当 $k$ 很小时（$k < 5$），增加一个簇能大幅降低失真（$D(k)$ 快速下降）；当 $k$ 很大时（$k > 30$），增加簇的边际失真改善趋近于零。人类选择的 $k=18$ 恰好落在这个\textbf{边际收益递减}的过渡区域。从信息论的角度，这个拐点对应率失真函数 $R(D)$ 的\textbf{斜率} $-\frac{dR}{dD}$ 开始快速增加的区域——即再增加率（更多的簇）只能换来极少的失真改善。

\subsection{直观理解}

信息瓶颈原理认为，一个好的表示应该在\textbf{压缩}（减少信息量）和\textbf{意义保持}（保留对下游任务有用的信息）之间取得平衡。用聚类来理解：

\begin{itemize}[noitemsep]
  \item $k=2$：把所有数据压成 2 个簇——极度压缩但失真巨大（$A$ 和 $B$ 被强行合在一起）
  \item $k=50$：拆成 50 个细类——几乎没有压缩，每个簇内的数据非常相似
  \item $k=18$：人类定义的 18 个类别——这是一个自然的折中点
\end{itemize}

$\mathcal{L}$ 量化了这个权衡：
\[
\mathcal{L}(k) = \underbrace{I(X;C)}_{\text{复杂度}} + \beta \cdot \underbrace{D(k)}_{\text{失真}}
\]

直观上：左边第一项 $I(X;C)$ 正比于 $\log_2 k$（更多簇需要更多信息来描述），第二项 $D(k)$ 随 $k$ 增大而减小（更多簇使每个簇内部更紧凑）。$\mathcal{L}$ 曲线呈 U 形还是单调递减，取决于数据的几何结构。在我们的数据中，$\mathcal{L}(k)$ 整体随 $k$ 单调递减但斜率逐渐变缓——说明增加簇数带来的边际收益递减。人类的 $k=18$ 恰好落在这个"收益递减"的区域。

\subsection{方法}

\textbf{目标：} 检验 LLM 嵌入空间在不同粒度聚类下的压缩-失真权衡曲线，以及人类选择的类别数 $k=18$ 是否位于这条曲线的自然拐点附近。

\paragraph{输入：} 与 RQ1 相同的 $n$ 个概念的\textbf{静态嵌入} $\mathbf{e}_i \in \mathbb{R}^d$，分属 $k_{\text{human}}=18$ 个人类类别。

\paragraph{流程：}
\begin{enumerate}[noitemsep]
  \item \textbf{加载嵌入：} 读取所有概念的静态嵌入（来自 	exttt{models\_embeddings/\{model\}\_static.npy}），形状 $(n, d)$。
  \item \textbf{遍历 $k$ 值：} 对 $k = 2, 3, \dots, 50$ 分别运行 $k$-means（$n_{\text{init}}=50$），得到簇分配 $\{C_k\}$ 和质心 $\{\mu_{c}^{(k)}\}$。
  \item \textbf{计算压缩项 $I(X; C_k)$：} 对于 $k$-means 聚类，$C_k$ 是 $X$ 的确定性函数（每个点唯一分配一簇），因此 $H(C_k \mid X)=0$，压缩项简化为 $I(X; C_k) = H(C_k)$。$H(C_k)$ 根据簇大小分布计算：
  \[
  H(C_k) = -\sum_{c=1}^{k} \frac{|C_k^{(c)}|}{n} \log_2 \frac{|C_k^{(c)}|}{n}
  \]
  若 $k$ 个簇大小均衡，则 $H(C_k) = \log_2 k$。
  \item \textbf{计算失真项 $D(k)$：} 所有点到其所属簇质心的平均欧氏距离平方：
  \[
  D(k) = \frac{1}{n}\sum_{c=1}^{k} \sum_{\mathbf{e} \in C_k^{(c)}} \|\mathbf{e} - \mu_c^{(k)}\|^2
  \]
  \item \textbf{复合目标：} 计算 $\mathcal{L}(k) = I(X; C_k) + \beta D(k)$，固定 $\beta=1.0$。
  \item \textbf{记录人类基准：} 在 $k_{\text{human}} = 18$ 处记录 $\mathcal{L}(18)$ 供跨模型对比。
\end{enumerate}

\paragraph{输出：} 每个模型得到一条 $\mathcal{L}(k)$ 曲线（$k=2,\dots,50$），以及 $k=18$ 处的标量值 $\mathcal{L}(18)$。

\paragraph{计算量：} 24 个模型 $\times$ 49 个 $k$ 值 $\times$ 50 次初始化 = 58,800 次 $k$-means 运行。

\subsection{核心发现}

\textbf{发现 1（LLM 嵌入同质性）：} 22 个 Transformer 模型的 $\mathcal{L}$ 值集中在 $[4.15, 4.55]$——跨度仅 0.40。模型大小（0.5B $\to$ 32B）、架构（encoder $\to$ decoder）、家族（Qwen $\to$ GPT $\to$ Gemma $\to$ DeepSeek）、训练数据量对 $\mathcal{L}$ 的影响在测量误差水平以下。\textbf{所有 LLM 的静态嵌入在压缩-意义框架下几乎是等价的。}

\textbf{发现 2（经典嵌入完全不同）：} word2vec（$\mathcal{L}=12.13$）和 GloVe（37.93）高出 3 到 9 倍。直观解释：word2vec 的嵌入是 50 维的标准化词向量，每个词只有一个向量；LLM 的静态嵌入是 768-7168 维的上下文平均，每个词在不同句子中不同但被平均到同一个向量。后者的语义区分度更高但也更"平滑"，导致 $\mathcal{L}$ 更低。

\textbf{发现 3（人类的 $k=18$ 在规律性区域）：} 虽然 $k=18$ 并非严格的数学拐点，但它恰好位于所有 LLM 的 $\mathcal{L}$ 曲线从"快速下降"过渡到"缓慢下降"的区域。在这个区域，增加聚类数的边际收益开始急剧递减——人类选择的类别数正好是这个过渡区间的典型值。

\begin{itemize}[noitemsep]
  \item \textbf{AMI} 衡量聚类结果与人类标签的\textbf{对齐程度}（语义层面的匹配）
  \item \textbf{$\rho$}（RQ2）衡量嵌入相似度与人类典型性排序的\textbf{一致性}（类别内细粒度匹配）
  \item \textbf{$\mathcal{L}$} 衡量嵌入空间的\textbf{几何压缩属性}（信息论层面的结构）
\end{itemize}
AMI 与 $\rho$ 的中等正相关（$r=0.619$，仅 LLM）是预期的：两者都反映了嵌入空间与人类认知结构之间的匹配程度，只是粒度不同（类别边界 vs 类别内排序）。$\mathcal{L}$ 与另两者几乎无关，进一步确认了 $\mathcal{L}$ 衡量的并非\"对齐\"，而是嵌入空间的固有几何属性。

\section{限制与讨论}

\begin{enumerate}[noitemsep]
\item \textbf{仅 McCloskey1978 数据集：} 论文使用了 3 个数据集，我们只完整复现了 McCloskey1978（$\sim$200 个概念，18 个类别）。BERT/RoBERTa 在其他数据集上的 AMI 可能会更高。RQ2 的结果也受此限制——Rosch1973（48 条）和 Rosch1975（565 条）的数据量影响了统计显著性。
  \item \textbf{$n_{\text{init}}=100$ vs $n_{\text{init}}=10$：} 提高 $n_{\text{init}}$ 仅轻微改善结果（高度数集上提升 $<0.01$ AMI），说明论文的 $n_{\text{init}}=10$ 足够稳定。
  \item \textbf{缺少 16 个模型：} 论文使用了 40+ 个模型，我们只能复现其中 24 个（缺少 Llama、Mistral、CLIP 等）。不过由于 LLM 的 $\mathcal{L}$ 高度收敛，预计缺失模型不会改变定性结论。
  \item \textbf{$\mathcal{L}$ 对 $\beta$ 敏感：} 我们固定 $\beta=1.0$。$\beta$ 的不同取值会改变 $\mathcal{L}$ 的绝对数值，但跨模型的相对排序应保持稳定。
  \item \textbf{信息论假设的简化：} 我们将 $I(X;C)$ 近似为 $\log_2 k$，并直接用 $D(k)$ 替代 IB 中的 $-I(C_k;Y)$。更严格的做法是使用 Blahut-Arimoto 算法直接计算率失真函数。但由于 $k$-means 是率失真求解的一种实际近似，且我们的主要目的是跨模型比较，这些简化是合理的。
  \item \textbf{数据处理不等式的补充：} 根据 DPI（式~8），$I(X;Y) \geq I(C_k;Y)$。因此，聚类结果与人类标签的互信息不可能超过原始数据与标签的互信息。这为我们解释为什么 LLM 的 AMI 普遍较低提供了另一个视角：如果原始嵌入 $X$ 本身含有的类标签信息本就有限（由于句法和语用噪声），那么进一步的聚类只能保持或减少这些信息。
\end{enumerate}
